Evaluations help

The Evaluations area of the SuccessKPI platform is designed to give you a quick and easy tool for call observations, listening, and quality management and analysis.

From Evaluations, you can navigate back to Home, Analytics, Playbooks, and Settings with the top menu. To work within the Evaluations area, use the side navigation for Evaluation workspace, Evaluation forms, Interactions, and this Help tab. You can collapse the side navigation with the side arrow at the top of the gray panel.

Evaluation workspace

You first arrive in the Evaluation workspace which acts as an inbox for your conversation evaluations. The top panel is your selection area. The bottom section is your work area. If there are no evaluations in your Evaluation workspace, you can put evaluations here by clicking on the Interactions tab or by building a Playbook with an action to push calls to a specific Evaluation forms.

Sort Interactions

In the Sort interactions area, you can select a specific Evaluation form. The area is set up by “forms” so that you are always working on one specific type of QA work. To change the form (and related calls) used for evaluation, access the pull-down menu “Evaluation form applied.” Once selected you can further refine your search by date, by Agent name, or by Queue. Click Refresh results to refine your search. The conversations that meet your selection criteria then appear in the “Select interactions for evaluation” area in the right side of the top panel. You can select any call in this selection area and it will then appear for evaluation below in the Interaction Player and the appropriate evaluation form will appear ready for completion.

Interaction player

The interaction player allows you to see and hear conversations at the same time. You can pause, start, and move around in the call as needed. The media player shows the transcript for the related call under the Transcript tab located below the Interaction player. The Topics tab allows you to see key topics and key phrases including where they appear in the call. You can “jump” to the related portion of the conversation by clicking on the keyword.

Evaluation form

The Evaluation Form on the right-hand side shows the set of criteria and questions that have been established by you and your management team for this type of call / conversation evaluation. As you work through the call listening, you can provide your scores for each question and hit save when you are done. Saved evaluations become available for reports and dashboards for agents, supervisors, and other reporting needs.

Evaluation forms Tab

The Evaluation forms tab allows you to create custom forms for use in evaluating conversations. Here you can create forms with categories and questions that you will use as the basis upon which to coach, train and develop your team.

Within the Evaluation forms area, you can select and edit existing evaluation forms or “Create a form” from scratch. You can also search for a form using the search box. Inside each evaluation form you will find you can edit:

  • Form Name (44 characters)
  • Description (70 characters)
  • Categories
  • Questions

Each form is set up by category area. You can associate categories directly with Themes (See Playbooks help) used in the Playbook area or create categories of your own. You can delete categories by clicking the trash can icon. Once you add a category you can give it a name. You can have one category or multiple categories.

Each category can have multiple questions. You can add as many questions as you like under each category. Once you add a question, you can type in the question and then select the response type. Once The response type is declared you may add point values to the potential answers by adding to the Answer values section of the question.

Questions can be of the following types:

  • Number: Allows you to create a range of choices for the user (e.g. 1-10)
  • Choice: Lets you create a menu of possible choices (e.g. Red, Yellow, Green for colors)
  • Yes/No: Allows only Yes or No responses (Yes, No)
  • Comments: Designed for free form text (Any text is permitted)
Interactions

There are two ways to get interactions into the Evaluation workspace: Playbooks and this Interactions tab. This tab is an active search tool that allows you to explore conversations and to trigger an action to push the calls into the appropriate evaluation workspace.

The search tool allows you to select calls by date range, call queue, agent name, topic (See: Playbooks), key phrase (see playbooks).

From there you can select a specific Evaluation Workspace and then click “Send to Evaluation Workspace” and this call will be placed in the Evaluation room.Calls can also be automatically sent to an evaluation workspace by creating Playbooks to move them on a per call basis immediately after the conversation is logged to the data warehouse.
